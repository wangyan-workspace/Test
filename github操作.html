git的使用

过一边流程

自己跟自己玩

1.在GitHub上先创建新的项目
2.将自己新创建的项目保存在本地 -> 找到某个想要保存项目的磁盘
3.右键 ->git Bash Here 之后输入命令行 git clone + 自己在GitHub上的地址（不能用ctrl+v，不好使，右键paste就将自己创建项目的地址粘贴到命令行中） 回车 等待…… Unpacking objects: 100% (3/3),出现这个标志，说明克隆成功
4.这时会在磁盘中出现GitHub上创建的项目文件夹 点击进入通过vscode打开，即可进行操作
5.保存文件，将更新的代码提交到远程仓库中
               ||
               ||
            \    /
              \/
6.在该文件夹的终端中打开(每次更新都要进行一遍这个操作)
7.git add . -> 回车 =>提交所有文件到暂存区
8.git commit -m '注释' -> 回车 =>方便以后查找不同状态的代码
9.git push -> 回车 =>将文件提交到远程仓库
10.在远程仓库刷新界面即可看到更新后的代码

自己跟别人玩

从GitHub上克隆其他人的代码（放到自己的磁盘（本地）里，方便查看）
开始->运行->cmd 或者
1.在某个确定的想要保存项目的磁盘里，右键 ->git Bash Here之后正常输入命令行（***推荐***） 如果打开是在其他目录里，可以通过cd ..寻找到上一级目录（最好放在D盘或者E盘）
2.确定指定的磁盘后，输入命令行 git clone +对方的项目地址（不能用ctrl+v不好使，右键paste就将对方的地址粘贴到命令行中）
3.克隆只针对第一次之后想获取对方的代码，在之前克隆的文件夹下， 右键 ->git Bash Here可通过git pull的方式拉取对方的代码
4.打开文件夹里的文件即可看到更新后的代码
